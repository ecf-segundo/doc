           Descobrir usuário que está conectado no computador Windows

   5 de março de 201525 de janeiro de 2018 - by admin

   Comando no prompt para descobrir quem está logado em determinado
   computador na rede, muito útil em empresas com domínio Windows. O código:

   >> wmic /node:nomedocomputador computersystem get username

   Colique o nome do computador que deseja descobrir quem está logado.

   descobrir-usuario-logado

   Este teste foi realizado em um ambiente de Domínio Active Directory com
   usuário administrador do domínio.
