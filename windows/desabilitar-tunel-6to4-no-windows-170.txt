                       Desabilitar túnel 6to4 no Windows

   15 de janeiro de 201825 de abril de 2018 - by admin
   IPv6

   Abrir o prompt de comando como administrador e digitar o seguinte comando:

     netsh interface ipv6 6to4 set state disabled

   Para verificar o Status do túnel:

     netsh interface ipv6 6to4 show state
