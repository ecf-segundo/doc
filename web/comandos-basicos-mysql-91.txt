                             Comandos básicos Mysql


   – Realizar login no mysql:
   mysql -u usuario -p

   – Exibir os bancos disponíveis:
   mysql> show databases;

   – Para visualizar as tabelas de uma database:
   mysql> use nome_banco;
   mysql> show tables;

   – Visualizar estrutura de uma tabela (ex: id, nome, endereço):
   mysql> desc nome_tabela;
