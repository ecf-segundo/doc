                 Acessar Arquivos do Iphone pelo terminal Linux

   29 de março de 202029 de março de 2020 - by admin

   Precisei transferir fotos e videos de um Iphone para outro pelo Linux e
   quando plugava o aparelho simplesmente não acontecia nada. Então dando uma
   googlada chequei na seguinte solução:

   Instala os seguintes pacotes (estava usando Debian 10)

     sudo apt install ifuse
     sudo apt install libimobiledevice6 libimobiledevice-utils
     sudo ifuse /mnt/
     sudo ls /mnt

   Assim pude copiar os aquivos de /mnt/DCIM/ para meu computador e depois
   transferir para o novo iPhone. Detalhe, não consegui abrir pelo
   gerenciador de Arquivos, então fiz tudo pelo terminal mesmo.
