               Usando o rclone para mountar Google Drive no Linux

   8 de novembro de 20218 de novembro de 2021 - by admin

   Usando o rclone para mountar Google Drive no Linux. Considerando que já
   tenha feito a configuração de acesso do rclone com o Google Drive.

   Use o seguinte comando para mountar

     mount rclone mount gdrive: /mnt/gdrive

   Use o seguinte comando para desmontar:

     mount fusermount -uz /mnt/gdrive
