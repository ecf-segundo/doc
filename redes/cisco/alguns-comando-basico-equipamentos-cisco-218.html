Alguns comandos básicos de equipamentos Cisco

12 de março de 201813 de março de 2018 - by admin

Alguns comando de configuração inicial de equipamentos da Cisco, sua forma resumida e um resumo do que cada um faz:

Comando Forma resumida O que faz
Router> enable Acessa o módulo privilegiado
Router# configure terminal conf t Acessa o modo de configuração Global
Router(config)# hostname R1 host R1 Altera o hostname do dispositivo
R1(config)# no ip domain-lookup Desativa relolução de nomes
R1(config)# banner motd “Acesso restrito” ba m “Acesso restrito” Desativa relolução de nomes
R1(config)# enable secret minhaSenha ena s minhaSenha Coloca uma senha para acessar o modo privilegiado
R1(config)# service password-encryption Criptografa as senha
R1(config)# line vty 0 15 Acesso o modo de configuração de linha para acesso remoto
R1(config-line)# password senhaRemoto pass senhaRemoto Coloca uma senha para acesso remoto telnet
R1(config-line)# login Permite o login remoto
R1(config-line)# exec-timeout 60 0 Temo que a sessão remota expira(60 minutos e zero segundo)
R1(config-line)# logging synchronous Desabilita mensagens de log evitando que apareçam na tela
R1(config)# line console 0 Entra no modo de configuração da console 0
R1(config-line)# password senhaConsole pass senhaConsole Coloca senha para acesso via console
R1(config-line)# login Permite o login via console
R1(config)# interface gigabitEthernet 0/0 int g0/0 Entra no contexto de configuração da interface
R1(config-if)# ip address 192.168.0.1 255.255.255.0 ip addr 192.168.0.1 255.255.255.0 Coloca um ip na interface do roteador
R1(config-if)# no shutdown no shut Ativa a interface
R1(config)# end Vai direto para o modo privilegiado
R1# copy running-config startup-config copy run start Salva configurações
R1# copy running-config startup-config Vai direto para o modo privilegiado
R1# write memory wr Também salva configurações
R1# show running-config sh run Exibe configurações atuais
R1# show startup-config sh start Exibe as configurações salvas
R1# show ip interface brief sh ip int br Exibe configurações resumidas de interface
R1# show ip route sh ip route Exibe a tabela de roteamento
